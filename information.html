Info

What This Is

This website is designed, programmed, and hosted by Noah Kirby (me) as a way to host my self programmed minimalist version of tetris (WIP), as well as any other projects I may want to share, or any tests I feel like posting.

Hardware

This website is being hosted on my home pc on a VirtualBox VM with T-Mobile Home Internet. It's limited to 3gb of ram and 20gb of memory with 1 cpu core. Thus, limited performance will not be an issue

Software

The server itself is running on Apache2 for Ubuntu. All port fowarding and the 'domain' assignment are handled by Pinggy, a paid multi purpose proxy service. The files are stored locally on the VM, and remotely on a Git Repo.
