Info

What This Is

This website is designed, programmed, and hosted by Noah Kirby (me) as a way to host my self programmed minimalist version of tetris (WIP), as well as any other projects I may want to share, or any tests I feel like posting.

Hardware

This website is being hosted on a $15 Raspberry Pi Zero 2 Wireless stored in an altoids tin in my backpack. It's powered by a 20,000mah usb battery pack and recieves wifi from my phone's mobile hotspot.

Software

The server itself is running on Apache2 for linux. All port fowarding and the 'domain' assignment are handled by Pinggy, a paid multi purpose proxy service. The files are stored locally on the Pi, and remotely on a Git Repo.
